# v0.18.0

Full merge of upstream cablehead/xs (through v0.13.5-dev) into the joeblew999
fork. This pulls the fork up to the current engine and store feature set while
keeping the fork's additive changes (XS_RELAY_URL relay mode, mise/CI tooling).

## Highlights

- Nushell 0.113.1 (was 0.112.1); MSRV raised to 1.93.1 (rust-toolchain.toml).
- Multi-topic pattern reads: `.cat` / `.last` accept a Nushell list of topic
  patterns; reads accept multiple comma-separated patterns.
- Prepared-engine fixes: `.append` works on restart and inside modules.
- A Store can carry a base engine that processor engines clone.
- Actor loop runs on a dedicated thread with inline eval.

## Fork notes

- Versioned 0.18.0 in the fork's own scheme (upstream is at 0.13.5-dev).
- Consumed by joeblew999/http-nu via git tag.
